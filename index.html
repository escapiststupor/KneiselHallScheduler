Rumor has it that a leading chamber music festival always comes with leading technology...

This is a complete remake of the scheduling software based on Melody Chan's prototype with a newly built UI so everyone can use it, even if you are very far from being geeky.

In short, this program takes in all the necessary information and automatically generates scheduling tables that you can either print directly from the browser or copy paste to a text editor (ex: MS word or OSX Pages).

Please follow the instructions until the very end and DO NOT REFRESH THE PAGE as you go along. If you do so you might have to go over from the very start again. Make sure all your inputs are correct before you press 'move on'.

by Phyllis Yen. KH alum 2015.

*Note: The order of input matters. If, say, a string quintet is played by Mia Dietrich, Alice Hong, violins; Rachyl Duffy, Mei-Chun Chen, violas; Ye Young Yoo, cellos; please click on the names according to the order of appearance, i.e. click first on Mia Dietrich, and then Alice Hong, and then Rachyl Duffy, and then Mei-Chun Chen, and then Ye Young Yoo. That's the only way this program knows Mia is on first violin and Alice is on second violin, etc etc. The program is not yet smart enough to know who is a fit for first violin and who is not...
